  1. Introduction

    Hadoop is open-source distributed framework used to process big data. It is written in Java, with some native C and shell script command line tools.

    Hadoop relies on clusters of commodity hardware, helps handle failure. (Hadoop was demonstrated to work with 2000 nodes)

    The core of Hadoop contains Hadoop Common (utilities), Hadoop Distributed File System (HDFS), Hadoop YARN (job scheduler), Hadoop MapReduce (task runner), and Hadoop Ozone (object store, a new subproject)

    Inspired by Google File Systems and MapReduce

    Operation Modes

    Hadoop has three operation modes

    1. Local/Stondalone Mode: (default), run as a single java process
    2. Pseudo Distributed Mode: run serveral java processes in a single machine, (good for development and debugging)
    3. Fully Distributed Mode: run on serveral machines (nodes)

  2. Installation

    Current (2018.10.7) stable version 2.9.1

    Setup a single node cluster

    1. Prerequisites: Linux, Java, sshd service is running (manage remote Hadoop daemons)
    2. Download the hadoop here. And assign the java path to the etc/hadoop/hadoop-env.sh 
                              # the value depends on your system
                        # JAVA_HOME is the home of the bin directory of Java
                        # for example, if the java is located at /usr/bin/java, then the value here is just /usr
                        export JAVA_HOME=/usr
      Test by run ./bin/hadoop
                              add to path .bashrc
    3. To run a hadoop task, we need to define a task (a .jar), input (files), output (a directory) and optional arguments. 
                              # hadoop provides examples of jar. The following one is to count the 
                        # prepare input
                        mkdir input
                        cp etc/hadoop/*.xml input 
                        # run the task
                        hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-2.9.1.jar grep input output 'dfs[a-z.]+'
                        cat output/*
