    C#'s class has some new features compared with Java. But most of these features are provided by C++.

    2. Operator overload

    6. Nested Types (Java inner class)

    Modifier for accessibility

    1. public: all
    2. protected: self and drived class
    3. internal (default): access limited to the current assembly (assembly is a .dll library or a .exe, it is not a namespace. (different from java's package))
    4. protected internal: protected internal has a more border accessibility than internal. It is like protected || internal. 
                              // assembly1.dll
                            public class BaseClass
                            {
                               protected internal int myValue = 0;
                            }
                    
                              // assembly2.dll
                            public class SubClass:BaseClass{
                                public static void Main(){
                                    BaseClass bc = new BaseClass();
                                    // bc.myValue // error since different assembly
                                    SubClass a = new SubClass();
                                    a.myValue; // correct since drived class.
                                }
                            }
    5. private: inside the class
    6. private protected: subclass can access private field if the subclass and baseclass in the same assembly.
    Rules
    1. Default access modifier of class is Internal. And private for class member.
    2. Subclass's accessibility < Parent class e.g. Subclass Public and Parent class Internal is an error.
