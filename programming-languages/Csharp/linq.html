  1. Introduction

    LINQ (language integrated query) [pronunciation: link] is a C# language feature that supports query data collections (C# list, database). Different from other database providers e.g. Npgsql that implemented as a library, LINQ is a language feature built into C#. It is a first-class language construct, just like classes, methods, events.

    Support: C# 3.0 and later

    1. LINQ-enabled data source

      A linq-enable data source means the data type can be queried with LINQ. A data type that implements IEnumerable<T> or IQueryable<T> is LINQ-enabled.

    2. LINQ provider

      For data sources that are not in the C# object, a linq provider is needed to map the data source into the C# LINQ-enabled data source.

                          XElement contacts = XElement.Load(@"c:\myContactList.xml"); // load xml

    3. Query Syntax vs Method Syntax

      LINQ provide two syntax, Query syntax (like SQL) and method syntax (a chainable method call). When executing, the query syntax is first compiled into method syntax and then executed.

                      // Query syntax
                IEnumerable<int> numQuery1 = 
                            from num in numbers
                            where num % 2 == 0
                            orderby num
                            select num;
                //Method syntax:
                IEnumerable<int> numQuery2 = numbers.Where(num => num % 2 == 0).OrderBy(n => n);

      Some query operations, such as Count or Max, have no equivalent query expression clause and must therefore be expressed as a method call.

  2. Examples

    1. Simple

      A LINQ uery operation involves three parts: an enumerable data source, an linq expression, executing linq expression

      A LINQ query return a IEnumerable<Type>, but var is used in most case.

      Query is executed when foreach

                                  // Define data source
                            int[] scores = new int[] { 97, 92, 81, 60 };
                            // Define the query expression.
                            IEnumerable<int> scoreQuery =
                                from score in scores
                                where score > 80
                                select score;
                            // Execute the query.
                            foreach (int i in scoreQuery)
                            {
                                Console.Write(i + " ");
                            } 


                            var scoreQuery_var =
                                from score in scores
                                where score > 80
                                select score;
                                            // Execute the query.
                            foreach (var i in scoreQuery_var)
                            {
                                Console.Write(i + " ");
                            }

    2. Aggregation

                                  var scoreQuery =

                                select score;
                            scoreQuery.Min(); // .Count(), .Max() ...
