  1. Introduction

    1. Delegate

      C# supports a kind of type called delegate, which is basically function pointer. A delegate type defines a signature of a function, including return type, parameters' types, and parameters' order.

      Delegate belongs to reference type, it is on the same level as Class, Interface, and Struct.

                          delegate void f1(int n, string str);
                    delegate void f2(string str, int n);
                
                    static class Utility { 
                        public static void print(int n, string s)
                        {
                            Console.WriteLine($"{s}: {n}");
                        }
                    }
                    class Program
                    {
                        static void Main(string[] args)
                        {
                            f1 fun = Utility.print;
                            fun(10, "Ten");
                        }
                    }

  2. Lambda

    Lambda is a anonymous funcion, (thinking about the immediate value), which can assigned to a delegate variable.

    1. Expression Lambda

                              // (input-parameters) => expression
                        // the expression result is returned automatically.
                        delegate int f(int i);
                        f fun = i => i + 1;

    2. Statement lambdas

                              // (input-parameters) => { statement; }
                        // have to manually return result.
                        delegate void f2(string str, int n);
                        f2 fun2 = (string str, int n) => { Console.WriteLine(str + n); };
