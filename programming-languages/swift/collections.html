  1. Introduction

    Swift defines list, set, dictionary (hashmap). There is no array even though the "list" is called array.

  2. Array (List)

    Swift's array is a list. The size of the array is mutable.

                
    var intArr: Array<Int> = Array<Int>()
    var simpleIntArr: [Int] = [] // syntax sugar, [Int] == Array<Int>
    var simpleIntArr2 = [Int]()
    var numbers = [1,2,3] // type inference with non-empty array.
