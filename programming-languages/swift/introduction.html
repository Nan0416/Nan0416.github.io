  1. Introduction

    Swift is general-purpose programming language developed by . Its development started in July 2010, and the first version 1.0 was in 2014. The current (2019.11) version is 5.1.2

    Swift is a multi-paradigm (object-oriented), compiled programming language.

    Special Features

    1. Protocol-oriented programming (interface/agreement): extension based on protocol. (like C# extension methods).
    2. Value type: support not only class, but also struct. (different from java, same as C#)
    3. No primitive type: int, double are value type (struct).
    4. Support closure.
    5. Optional and chaining.
    6. Type inference: 后置type
    7. Functions are the first-class member. (different from java).
    8. Garbage collector: automatic reference counting (ARC).
    9. let & var: let makes a constant, var makes a variable. This constant restricts the content of value type and the reference of reference type.
    10. switch support break use fallthrough instead of break

  2. Syntax

    Features:
    0). no ; semicolon
    1). type inference
    2). string interpolation
    3). double-quoted string
    4). statement, function invocation can be written in global, no main function is required (kind of python) 

    
    // declare variable
    // {let | var} variable_name [: type] = value
    let age = 25
    let weigth: Double = 144

    // string interpolation, string is double-quoted.
    let intro: String = "I am /(age) years old"
    print(intro)

    // let assignment.
    // a let variable can be assigned after the declaration, but only assign once and cannot use it before the assignment.
    let message: String
    // print(message) cannot use before assigning value.
    message = "hihi"
    print(message)

  3. "primitive types"

    Swift doesn't have primitive types like C/C++ and Java. Numeric type and boolean type are struct, so they start with a capital letter.

    1. Integer: swift provides 8,16,32,64 bit signed and unsigned integers.

      Int8, Int16, Int32, Int64, UInt8, UInt16, UInt32, UInt64

      Int/UInt: a signed/unsigned integer whose size is same as the platform native word size. 64-bit machine is Int64/UInt64.

  4. Control Flow

    No parenthese are needed around the condition

    Must have {} even if one line of code. curly brace {} must

    1. If else

                              
    let small = 10, big = 20, boo = true
    if small < big{
        print("small")
    }else if boo{
        print("boo")
    }else{
        print("else")
    }

      guard "early exit"

      guard extract the value from optional.

    2. while loop

      while condition { statements }
      repeat { statements } while condition 

      
        var count = 0
        while count < 3{
            print("while loop")
            count += 1
        }
        count = 0
        repeat{
            print("repeat while loop")
            count += 1
        }while count < 3

    3. for in loop

      Because no ; in swift, there is no for(;;) loop.
      for i in 1...5 {} // both sides inclusive. i is by default const. 

      
    for var i in 1...3{
        i += 1
        print(i, "for-in")
    }
    for i in 1...3{
        //i += 1 // by default i is "let" const.
        print(i, "for-in")
    }
    for _ in 1...3{
        print("for-in")
    }
