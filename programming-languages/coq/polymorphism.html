  1. Introduction

    Polymorphism makes type as a parameter, but this parameter can be used in type's position.

    1. Example

                      Inductive natlist: Type :=
                | nil
                | cons (n: nat) (l: natlist).

                Inductive list (X: Type) :Type := (* inductive definition usually doesn't need a parameter. but now it can accept a type parameter.*)
                | nil
                | cons (n: X) (l: list X). (* inside the definition, X is used as a type. *)

                Definition mynatlist : list nat := cons n nil.

    2. Definition function for the generic list

      The type parameter should be placed at the beginning since we will use it in the later argument.

                      Fixpoint length (X: Type) (l: list X): nat :=
                match l with
                | nil => O,
                | cons _ l' => S (length X l')
                end.

    3. Implicit type parameter in function

      Coq can infer the type parameter according to the input.

                      Fixpoint length {X: Type} (l: list X): nat :=
                match l with
                | nil => O,
                | cons _ l' => S (length X l')
                end.

      When applying length with giving a list of a certain type instaces, coq can infer the type.

      You can also supply the type by using @

                      @length nat l. (* in this case, @length must be given the type. *)

    4. higher-order function

      A function has another function as input

                      Fixpoint natlistfilter (predicate: nat -> bool) (l: natlist) : natlist :=
                match l with
                | nil => nil
                | cons n' l' => if (predicate n') then cons n' (natlistfilter predicate l') else (natlistfilter predicate l')
                end.

                Fixpoint filter {X: Type} (predicate: X -> bool) (l: list X): list X :=
                match l with
                | nil => nil
                | cons n' l' => if (predicate n') then cons n' (natlistfilter predicate l') else (natlistfilter predicate l')
                end.
