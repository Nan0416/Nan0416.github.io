  1. Function

    Python has different ways to define and provide function parameter.

    1. General Rules

      Argument order: parameters without default value > parameters with default value > a tuple parameter > a dict parameter

      keyword parameter call > positional parameter call

    2. Function call with keyword parameter

                      def fun(l, r):
                    print(l, r)    
                fun(r="right", l="left") #left right

    3. Function call with default argument values

                      def fun(l, r="right", m="middle"):
                    print(l, m, r)    
                
                fun(r="right", l="left") #left right
                fun('left')

    4. Function with *args, **kwds

                      def fun(l, r, m="middle", *args, **kwds):
                    print(l, r, m)
                    print('tuple', args)
                    print('dict', kwds)
                
                fun("left", "right", "middle", "1", "2", "3", t=20,dir="left")
                # left right middle 
                # tuple ('1', '2', '3') 
                # dict {'t': 20, 'dir': 'left'}

    5. Python 3.8 provides / and * to restrict parameter types.
