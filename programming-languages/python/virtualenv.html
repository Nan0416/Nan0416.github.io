  1. Introduction

    VirtualEnv is a tool used to manage python runtime version, it can create an isolated environment, including python version, dependencies.

  2. Examples

    1. Create and Launch a new environment.

      Don't need root privilege.

      						# create a directory, under that directory
						which python3.7 # get python3.7 path
						virtualenv [--python=/usr/local/bin/python3.7] envName;
						cd envName;
						source ./bin/activate

    2. Install dependencies

      Using pip to install dependencies. Either "pip install dep" or using requirements.txt file

      						pip install -r requirements.txt # from requirements file.
						pip install git+giturl # install from git
						pip install dependency-name # directly install.
