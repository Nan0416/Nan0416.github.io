  1. String manipulation

    String is immutable. Every time you modify a string, it acutally generates a new one.

    1. StringBuilder

                      StringBuilder sb = new StringBuilder();
                sb.append(* any type); // return a refernce to self so that can be chainable
                int len = sb.length();
                char c = sb.charAt(10);
                String result = sb.toString();

                constructor: (int init_cap), (String str)
                .append(String / char / char[]): self
                .insert(int offset, char / char[] / String): self. //Offset in terms of string
                .reverse(): self
                .substring(int s, int e):String
                .length(): int

    2. String

                      constructor: (char [])
                .charAt(int): char
                .toCharArray(): char []
                .equals(String): boolean
                .substring(int s, int e): String
                + (concate): String
                .length(): int
                .split(String): String[]
                .toLowerCase()/ .toUpperCase(): String[]
                .indexOf(String): int

    3. Practice

      1). Construct/Modify a string: if the size is known, then use char[] and new String(char[]). Otherwise, using StringBuilder.
      2). for loop a string: .charAt()
      3). upper case: (char)(char_c - 'a' + 'A') Return int

    3. Stack

                              Stack<String> st = new Stack<>();
                        st.push("example"); // different from "add" than append items, push/pop will add/remove the element at the first (addFirst). 
                        String value = st.pop();
                        String value = st.peek(); // return the top value but not pop.

    4. Queue

                          // With list
                    LinkedList<String> queue = new LinkedList<String>();
                    queue.addLast("String");
                    queue.addFirst("String");
                    String first = queue.get/peekFirst(); // no such element exception / null
                    String last = queue.getLast();
                    String first = queue.pollFirst();
                    String last = queue.pollLast();

    5. Heap (PriorityQueue)
