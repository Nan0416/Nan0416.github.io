  1. Services & Dependency Injection

    Logger, Configuration are services. A class can use these services by obtaining their instances, and instances are obtained by service injection.

    ASP.NET Core provides IServiceProvider, registering Service in "Startup.ConfigureServices" method. Service instances are initialized, given to class constructor, and destoried by ASP.NET core.

    Dependency chain

    A service can use another service, which results in a dependency chain.

    Service Abstraction

    A service usually abstract as an interface, and interface name is given to class's constructors.
    Interfaces are implemented by concrete classes. These concrete classes are registered to the service injection.

    Parameter's of A Service's constructor

    1. via configuration, 2. via explicity initialization.

    framework-provided service

    Service lifetime

    Longer lifetime services cannot use shorter lifetime services.
