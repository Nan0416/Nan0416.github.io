  1. Introduction

    Unity is a software frameworks for game development, it is also called a Game Engine.

    Unity features

    1. Unity is a game engine support 2d and 3d game development.
    2. Unity helps manage game resources, developers define game logic (writing code script)
    3. Unity support C# code.
    4. Unity can build the game and generate the source for different platforms, such as iOS, Android, Windows, Web, and Mixed reality devices, such as Microsoft Hololens and Magic Leaps.

    Job of Unity

    As a game engine, unity helps developers to render the scene, object projections, camera view, manage object lifecycles.
