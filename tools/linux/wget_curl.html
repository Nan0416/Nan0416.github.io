  1. wget

    wget is a web resource downloader. It supports HTTP, HTTPS, and FTP. Most requests sent with wget are GET.

    1. Syntax & Usage

                      # wget [OPTION]... [URL]...
                wget --output-document=123.txt http://nan0416.github.io # download the curl to 123.txt

      wget has several different kinds of options, including log, downloaded resource, http config, https config, ftp config and so on

    2. Log config

      Log by default print to stdout. But it can be muted or redirect to other location.

                      wget --quiet http://nan0416.github.io # without logging
                wget --output-file log.txt http://nan0416.github.io # output to log.txt

    3. Download config

      The output filename, download max speed, retries, username&password for ftp are configurable.

                      wget --output-document "note.txt" http://nan0416.github.io # name the output file
                wget --output-document - http://nan0416.github.io # download the file to stdout
                wget --limit-rate 20k http://nan0416.github.io # max speed is 20 KB/s

  2. Curl

    Curl can sent post request

    1. Syntax & Usage

                              # curl [options...] url

    2. Curl post

                              curl --request POST http://nan0416.github.io
