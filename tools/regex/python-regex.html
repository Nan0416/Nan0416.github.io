  1. Introduction

    Python3 provides regex under the "re" module. It provides 4 operations
    1). match: check if matching start at beginning of the string, return a match object.
    2). search: find a match inside the string (not necessary from beginning), return a match object.
    3). findall: find all the matched string, return in a list (longest string match)
    4). finditer: find all the matched string return a iterator of match object

    Backslash

    In python, \ is used to escaped special character defined in a string. For example, new_line = '\n', single_quote = '\''. But in regex, \ is also used to escape regex defined special character, such as \s matching space. However, because \s is not a python string defined special character, python string does not treat \s as a single character, instead, python string help escape the \ to \\. To avoid the unexpected escape, using python raw string.

    In a raw string, \ is just a backslash, does not escape any character.

  2. Example

    1. .match

                              import re
                        qin = re.compile("qin")
                        qin.match("nan qin") # return None, since match is not start from the beginning
                        qin = re.compile("n.*n")
                        match = qin.match("nan qin") # longest match, return a match object
                        match.group() # return the matched string 'nan qin'
                        match.start() # the start position of the string, .match always return 0
                        match.end() # the end position of the string.
                        match.span() # a tuple include start, end positions

    2. .search

      Same as .match but not necessary starting from beginning

    3. .findall

                              # longest match
                        qin = re.compile("q.*n")
                        qin.findall('qinnan qin') # return a list with one element ['qinnan qin']

    4. .finditer

                              iter = qin.finditer("string ....")
                        for match in iter:
                            pass

    5. Raw string

                              # Design a regex to match \snan, we cannot directly use regex '\snan' because \s is space.
                        # We have to escape the first \ in regex, it becomes \\snan. Now represent it in python string.
                        # In python string, \ is also a special character need to escape. A \ become \\, so the using python string represent the regex is "\\\\snan"
                        regex_string = re.compile('\snan') # match ' nan', python helps escape string special character \
                        regex_string = re.compile('\\snan') # string match ' nan', user escapes string special character \
                        regex_string = re.compile('\\\snan') # match '\snan', user escapes the first string specail \, python helps escape the second
                        regex_string = re.compile('\\\\snan') # match '\snan', user escapes the first string specail \

                        #using raw string, raw string does not have string defined special character
                        regex_string = re.compile(r'\\snan') # match '\snan'
