  1. Introduction

    Java String class provides 3 regex operations
    1). match: Test if an entire string matchs a regex, return boolean. 2). split: split a string by removing matched part. 3). replace

    Backslash

    In regex, \ is used to escape special characters, .e.g. \. makes . to just match . \s to make s match a space.
    But \ in java string is used to escape string defined special characters, e.g. \t tab, \n newline. So in order to escape a regex character, \ first will be escaped.
    e.g. \\. = \. = matchs anything.

  2. Examples

    1. match

                              "Hello. Hi?".matches(".ello\\.\\sHi\\?"); // . matches any; \\. only matches .; \\? only matches ?
                        "hello".matches("(h|H)(ello|i)");

    2. Split

                              "   wejoi  wjeio  qeoj     jiowe     ".split("\\s+");
                        =>["","wejoi","wjeio","qeoj","jiowe"] // the last empty string is ignored.
