  1. Introduction

    .NET standard defines System.Text.RegularExpressions namespace and related types. Both .NET core and .NET framework implement them.

    Operations

    .NET regex implement regex as static methods.

    1. Determine whether the regular expression pattern occurs. Regex.IsMatch
    2. Retrieve one or all occurrences of text that matches the regular expression pattern. Regex.Match / Regex.Matches
    3. Replace text that matches the regular expression pattern. Regex.Replace

    .NET regex

    .NET regex is defined as a string. When the regex need to use \, such as using \+ to match +, the string itself has to first escape \, which leads 

            string pattern = "\\+"; // => regex \+ => match +

  2. Example

    1. Assertion IsMatch

      // print all expressions that contains addition
using System;
using System.Text.RegularExpressions;

public class Example
{
    public static void Main()
    {
        string pattern = ".*\\+.*";
        string[] operations = { "1 + 2", "3 * 4", "1+2*3" };
        foreach (string operation in operations)
            if (Regex.IsMatch(operation, pattern))
            {
                /*
                 * 1 + 2
                 * 1+2*3                
                 */
                Console.WriteLine(operation); 
            }
    }
}

    2. Match/Matches

      A string may exist multiple matches to a pattern, and using Regex.Matches will return a MatchCollection.

                                  string pattern2 = "\\s\\+\\s";
                            string operation = "1 + 2 + 3";
                            MatchCollection matches = Regex.Matches(operation, pattern2);
                            foreach (Match match in matches) {
                                /*
                                * V: + I: 1
                                * V: + I: 5
                                */
                                Console.WriteLine("V:" +  match.Value + "I: " + match.Index);
                            }
                    
                              //longest match
                        string pattern = ".*\\+.*"; // the .* chooses all the match.
                        string operation = "1 + 2 + 3";
                        MatchCollection matches = Regex.Matches(operation, pattern);
                        foreach (Match match in matches) {
                            // V:1 + 2 + 3I: 0
                            Console.WriteLine("V:" +  match.Value + "I: " + match.Index);
                        }
