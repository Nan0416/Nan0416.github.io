  1. Introduction

    Microservice is an architecture for designing modern server-side application. It breaks down the monolithic design into individual, loosely coupled services (components). These services can be programed in different languages, scaled independently.

    Each service will have its own database

  2. Inter-Service Communication

    The communication between each component in a monolithic application is through method call. But in microservice architecture, different services run in separate processes, most likely in different machines. And they may be programed in different languages. So the communication between each services is through network.
