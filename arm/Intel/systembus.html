  1. Introduction

    System bus is used to connected the CPU with out onboard component, such as dedicated graphics card

  2. Progress

    1. (8-bit) ISA (Industry Standard Architecture): 1981 Intel 8088 IBM PC
    2. ISA: 1984, 16-bit, Intel 80286 IBM AT
    3. EISA (Extended ISA): 1988 32-bit, not successful
    4. PCI (Peripheral Component Interconnect): 1992 32/64-bit
    5. AGP (Accelerated Graphics Port): 1996, replaced by PCIe
    6. PCI Express: 2004

  3. PCIe

    PCIe is mainly developed by Intel to replace PCI, APG and PCI-X. It was released at 2004

    Features

    1. Serial communication, duplex communication
    2. Hot-swap functionality (热拔插)
    3. Communication channel is not shared

    Standards

    1. Each communication lane (channel) includes two "differential signaling" pairs, one receiving pair and one transimtting pair, thus one lane composed 4 wires
    2. Scalability: One physical interface can include 1 lanes, 2 lanes, up to 32 lanes. Represent by x1, x2, x4, x8, x16 and x32
    3. x1 and x16 are most common
    4. smaller size interface e.g. x4 device can be plugged into bigger slot, e.g. x16 e.g. x16 @ x4 mode.
    5. PCIe x1

  4. Storage Interface

    SATA

    SATA (Serial AT attachment) defines the physical interface (form factor) and protocols

    SSD -> NVMe -> M.2 -> PCIe 3.0 x4

    NVMe defines the protocol that allows the SSD directly attached to a PCIe

    M.2

    M.2 defines a form factor. M.2 supports both NVMe protocol and SATA protocol. It also support the Wifi card.

    M.2 connector types: A or E for wireless adapter, B or M for SSD. Different socket.
