  1. Introduction

    Display can be set to block, inline-block, inline.

    1. block: can have height/width. A block element always starts a newline. The default static position is on the left top.
    2. inline-block: can have height/width. It does not start with a newline.
      1). vertical alignment with other elements in the same line.
      2). default space between inline elemenets still exist
      3). It has padding, margin, border
    3. inline: cannot set height/width.
      1). default space can exist between any two inline element. 
                      <span>Hello</span><span>Hello</span> <!--HelloHello Hello Hello-->
                <span>Hello</span>
                <span>Hello</span>
      2). It has normal padding, border, margin-left,right, but it does not have margin-top,bottom.
      3). Because inline element does not have width, you cannot set the width of a text. e.g. 
                      <span>Hello</span> <!--width is unknown-->
      For some inline element, they may have built-in width,height .e.g. img

  2. Problem

    1. Problem: inline span message's width cannot be determined
      Solution: inline-block with setting width and text-algin: center
      Hello

      Explanation

      1). inline-block allows to set the width of span
      2). span becomes an inline-block element, but the inner text is always inline and it positioned top-left corner. Using text-algin: center to horizontal algin it to center
      text-algin: can algin any content, not just text, to the center within block or inline-block element.
    2. Problem: How to vertical algin text inside a inline-block span hello
      Solution: line-height = inline-block's height hello

      Explanation

      1). line-height can control the inner text's height (img's height). With setting line-height to the inline-block height, the text occupies all the height with putting it onto the center.

  3. Example

    1. Input and Button have default display: inline-block
    2. Button's box-sizing: border-box; which means height/width = padding + border + content
    3. Input's box-sizing: content-box: which means height/width = content
