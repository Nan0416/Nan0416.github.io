  1. Design Principles

    Types of data

    1. Univariate, Bivariate, Trivariate, Multi-variate (Hypervariate)
      tables, scatterplot, [scatterplot matrix, chernoff faces, mosaic plot, parallel coordinates, star plot (radar chart), star coordinate, table lens, crosstab, parallel set]
    2. Time series data
      Lines, Bump charts,
    3. Numerical vs. Categorical
      Cateogircal: shape, color, icon
    4. set data
      Venn Diagram <3 ; Euler Diagram > 3 , bubble set

    Focus of data

    1. Precise values
      Table, stem-and-leaf plot,
    2. Change over time (Time series)
    3. Distribution
      Boxplot

    Tips

    1. Don't use rainbow color for quantitative data
    2. Value of Vis: Time (save time to answer) + Insight (discover insight, e.g. relationship)

    Gestalt Laws (从心理学角度了解人类视觉上的天性）

    1. Proximity, Similarity, Closure,

    Concept

    1. Pre-attentive: 200ms
    2. Cluster at high dimensions: deal with large variates. e.g. k-means, hierachical clustering, aggregation
    3. Interaction: selection,filter, abstract/elaborate, encode, reconfigure, explore
    4. Dynamic Query: +work is faster, very natural interaction, support undo, exploration, reversing. - cannot express complicated logic,
    5. Static vs Dynamic time-series visualization:
      static: + show history, allow comparison
      animation gives feel for process & change over time.

      Tables vs Graphs
      Table: check and compare individual values, precise Value required
      Graph (encode value to shape): reveal relationships among values

    2. Symbolic Vis

      Use Size, Shape, Color, Grayscale, Orientation, Position to represent numbers.
      Use picture (iconic represenation) to represent categories. e.g. dog

      1. lines: connect value, show pattern and change over time
      2. bar: emphasizes individual values, comparing individual values
      3. (Turkey) box plot: show a distribution
