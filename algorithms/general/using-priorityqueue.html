  1. Introduction

    PriorityQueue guarantee a specific ordered queue (implemented with heap). It implement Java Queue Interface.

    1. Runtime

                      offer() O(lgn)
                peek() O(1)
                poll() O(lgn)
                iteration (Queue implements Collections, so foreach loop is supported.) // but the order is not guaranteed.

    2. Limitation

      You can iterate through a PriorityQueue, but you cannot randomly access an element or remove an element.

      If randomly accessing/removing is needed, consider using TreeMap/TreeSet

  2. Examples

    1. 218. The Skyline Problem

    2. 1046. Last Stone Weight

      Desc: given an array of positive integers, each integer is the weight of a stone. Now pick the two heaviest stone and smash them. if different weights, then the heavier stone has a remaining part, put the remaining portion back.

      Sol: classic PriorityQueue.
