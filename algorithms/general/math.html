    1. 96. Unique Binary Search Trees

      Desc: given n number 1, 2, 3 to n, the number of unique Binary Search tree can construct by using those n numbers.

      Sol: DP + math formula.
      G(n): the #BST with n numbers. G(0) = 1, G(1) = 1
      F(i, n): the #BST with n numbers and i as root. F(i, n) = G(i-1) * G(n-i)
      So G(n) = sum over i (G(i-1), G(n-i))
      Using DP to solve it.

    3. 1104. Path In Zigzag Labelled Binary Tree

      Desc:
      1104. Suppose you have a infinite binary tree, each tree node has two children, left and right. Each node is labeled with a number. The number is count from top to bottom in zigzag order.

                                          1
                            3            2
                        4       5     6      7
                      15 14   13 12 11 10  9  8
      Now, given a label, return the path from the root to the node. e.g. given 14 => [1,3,4,14]

      Sol:
      1104. depth starts from 0, then we have even depth order from left -> right, odd depth order from right -> left . create helper function: 1. findDepth(label), 2. findPosition(label, depth) 3. findLabel(position, depth). And then loop from bottom to top until depth = 0. We need two cache to store the layer size and total size on a depth.
