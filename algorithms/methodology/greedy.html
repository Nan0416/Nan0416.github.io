    1. 53. Maximum Subarray

      Desc: given an array of integers, find a continuous subarray that has the largest sum e.g. [-1, 4, -2, 4, -9, 8] -> 6 [4,-2, 4]

      Sol: greedy, one the a local sum is less than (or equal to) zero, than restart the sum.

    2. * 55. Jump Game

      Desc: given an array of non-negative integers, each integer representes the #steps it can jump from the current index. check if it can the last index. e.g. [3,1,2,0,4]: yes 3->2->4

      Sol: the intutive solution is to simulate the jump process with recursion.
      But it can be solved with greedy. loop the array, keep a "farmost". find if we can reach i+1 from i, if we can update "farmost" by testing i+1. 

                          int farmost = nums[0];
                    for(int i = 1; i < nums.length; i++){
                        if(farmost >= i){
                            farmost = Math.max(farmost, nums[i] + i);
                        }else{
                            return false;
                        }
                    }
                    return true;
