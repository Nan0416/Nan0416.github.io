  1. Introduction

    Backtracking is a kind of recursion, sub-recur is used to solve the same problem.

    还是有些像DFS. Difference is DFS operates on a tree or graph, backtracking usually operates on an array. AND backtracking need to keep track of the visited path.

  2. Examples

    Where to apply?

    1. Backtracking usually applied to string and array based question. AND the output is the enumeration of all valid result.

    Example

    1. 39. Combination Sum

      Desc: an array of positive integers without duplicated and a target number. find all unique combinations' sum = target
      [2,3,5], 8 => [[2,2,2,2], [2, 3, 3], [3, 5]]

      Sol: Backtracking. Supposed 2 is used, then [2,3,5] = 8-2 = 6. recursive this step until target less than 1 since all integer > positive.

                      for(int i = 0; i < arr.length; i++){
                    helper(i, arr, target-i);
                }
                helper(int from, arr, target){
                    // arr[from] must be used in result.
                    if(arr[from] > target){
                        return;
                    }
                    if(arr[from] == target){
                        add to result.
                    }
                    tempresult.add(arr[from]);
                    for(int i = from; i < arr.length; i++){
                        helper(i, arr, target - arr[from]);
                    }
                    tempresult.removeLast();
                }
