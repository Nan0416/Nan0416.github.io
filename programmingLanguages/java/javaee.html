  1. Introduction

    Java EE, Java 完全体

    1. Java SE is for small scale development, e.g. a desktop application. Java EE is for server-side large scale programming.

    2. Current version of Java EE is Java EE 8. (2018.9.30)

    3. Compared Java SE, Java EE not only provides more packages, e.g. javax, but also defines more standards.

    4. JSR (Java Specification Requests) is developed by the Java Community Process.

    Servlet Container and Servlet

    JSR-369 defines the standard of Servlet container and Servlet. Including lifecycle, common APIs.
